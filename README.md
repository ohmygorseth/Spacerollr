# Space Rollr

A fast-paced browser game where you guide a neon ball across floating platforms through space. Survive as long as you can — the track loops forever and the leaderboard never sleeps.

🎮 **[Play Space Rollr](https://ohmygorseth.github.io/Spacerollr)**

---

## How to Play

- Stay on the neon platforms — fall off and it's game over
- The ball speeds up over time, so stay sharp
- The track loops endlessly — your score keeps climbing
- Beat the world record!

## Controls

| Input | Action |
|-------|--------|
| ← → Arrow keys | Move left / right |
| Space | Jump |
| ESC | Back to menu |

---

## Features

- 🌌 Endless looping track built from 4 hand-crafted levels
- 🏆 Local leaderboard (your personal top 10)
- 🌍 Global leaderboard powered by Firebase (top 100)
- 🎵 Soundtrack that cycles through tracks automatically
- ⭐ Neon space aesthetic with star field and particle effects
