# Space Rollr

A fast-paced browser game where you guide a ball along floating neon platforms through space — dodge the gaps and survive as long as you can.

🎮 **[Play Space Rollr](https://ohmygorseth.github.io/ballzy)**

---

## Controls

| Input | Action |
|-------|--------|
| ← → Arrow keys | Move left / right |
| Space or ↑ | Jump |
| Enter | Start / Restart |

---

## How to Play

- Stay on the neon platforms — fall off and it's game over
- The speed increases over time, so stay sharp
- Reach the finish line to complete the level
