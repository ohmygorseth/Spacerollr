# Ballzy

A fast-paced browser game where you guide a ball along a floating track — dodge the gaps, grab fuel, and survive as long as you can.

🎮 **[Play Ballzy](https://ohmygorseth.github.io/ballzy)**

---

## Controls

| Input | Action |
|-------|--------|
| ← → Arrow keys | Move left / right |
| Space or ↑ | Jump |
| Space / Enter | Start / Restart |

Touch controls are available on mobile.

---

## Tile Types

| Tile | Effect |
|------|--------|
| 🔵 Normal | Standard tile |
| 🟢 Boost | Increases speed |
| 🔴 Slow | Decreases speed |
| 🟡 Fuel | Restores fuel |

---

## How to Play

- Stay on the track — fall off and it's game over
- Keep an eye on your fuel bar — if it runs out, the run ends
- Speed increases over time, so stay sharp
